Problem: Use features such as time of day, CO2 levels, temperature, humidity, etc to classify if a room is occupied or not

Feature Information:

Occupancy: 0 or 1, 0 for not occupied, 1 for occupied status
Hours: between 0 to 23, adjusted to be between 1 to 24
Minutes: between 0 to 59, adjusted to be between 1 to 60
Temperature: in Celsius 
Humidity: in percentage 
Light: in Lux
CO2 levels: in ppm 
Humidity Ratio: derived quantity from temperature and relative humidity, in kgwater-vapor/kg-air
